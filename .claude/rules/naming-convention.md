# Naming Conventions

**Trigger**: `**/*.tsx`

## Components
- Must use **PascalCase** (e.g., `Button.tsx`, `ThemeCard.tsx`).
- Exported component name must match content.
- Props interface should be `[ComponentName]Props`.

## Packages
- Must be scoped under `@crazyone/`.
- Theme packages: `@crazyone/ui-[theme-name]`.
